Lestim
======

##Test vala version:
**Copy xml file and install schema:**
```
# cp vala/data/org.lestim.gschema.xml /usr/share/glib-2.0/schemas/
# glib-compile-schemas /usr/share/glib-2.0/schemas/
```

**Compile and run:**
```
cd vala
make
./lestim
```
